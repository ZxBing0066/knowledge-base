# 盒模型

![picture 3](/img/box-model-68d7f077ea315054ee7680248d7926d9a592cb75d7934d0a03b3c83769456727.png)

-   内容区、padding、border、margin
-   怪异盒模型/IE 盒模型与标准盒模型的区别
    -   标准盒模型 content-box
    -   IE 盒模型 border-box
-   box-sizing 切换盒模型
-   块盒子和内联盒子区别
    -   内联盒子
        -   宽高被忽略
        -   margin、padding、border 生效
        -   不会撑开盒子的实际占用大小，影响外部布局
-   内部显示类型和外部显示类型
-   margin 折叠 - 垂直边距
