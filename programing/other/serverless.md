# serverless

-   一种云原生的开发模型，让开发不需要关注和管理服务器，只需要关注代码运行。
-   一般会按需响应和执行，比如常见的 nodejs serverless，当检测到路由指向对应的 nodejs 处理模块时将该模块调用并进行执行，
    平时代码不会运行。
-   一般情况下为按需付费

## 常见的 serverless 服务

-   firebase functions
-   vercel 支持 next.js 和 node.js 模式的 serverless
-   cloudflare workers
-   各个云厂商如阿里云、腾讯云、AWS 都有自己的 serverless 服务

大部分都存在免费额度，可白嫖。
