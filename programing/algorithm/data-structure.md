# 数据结构

## 链表

```ts
class LinkedListNode {
    value: any;
    next: LinkedListNode | null;
    constructor(value, next) {
        this.value = value;
        this.next = next ?? null;
    }
}
```

## 双向链表

```ts
class LinkedListNode {
    value: any;
    next: LinkedListNode | null;
    constructor(value, next) {
        this.value = value;
        this.next = next ?? null;
    }
}
```
